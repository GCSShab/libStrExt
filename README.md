# libStrExt  
Librairies C pour la manipulation de chaînes de caractères sous forme de librairies "linkable" dynamiquement et statiquement.   


compiler les librairies et le test : 
``` bash
./make 
```  
Tester la librairie so
``` bash
./test 
``` 

installer les librairies  
``` bash  
make install
```  

Supprimer les librairies  
``` bash  
make clean
```
